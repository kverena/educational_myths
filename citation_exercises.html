4
Reference Management in Practice / Literaturverwaltung
Import Pashler et al. (2008) into Citavi or Zotero using the DOI, set the correct citation style, and compare the generated entry with your corrected version from Task 3.
Importiere Pashler et al. (2008) per DOI in Citavi oder Zotero, stelle den richtigen Zitationsstil ein und vergleiche den generierten Eintrag mit deiner korrigierten Version aus Aufgabe 3.
~ 7 min
  1. Open Citavi (citavi.com) or Zotero (zotero.org) — both are free and available at Universität Passau.
  2. Import Pashler et al. (2008) by using the DOI and complete the entry if needed.
  3. Set the citation style to APA 7th edition.
    In Citavi: Citation → Citation Style  ·  In Zotero: Edit → Preferences → Cite → Styles
  4. Generate the reference list entry and copy it to clipboard.
    In Citavi: select reference → Copy as formatted citation  ·  In Zotero: right-click → Create Bibliography from Item → APA 7th → Copy to Clipboard
  5. Paste the generated entry below and compare it carefully with your corrected version from Task 3.
